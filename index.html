Это позволяет администратору легко управлять контентом на сайте и представлять новые работы художника миру.

Авторизация

Для обеспечения безопасной авторизации администратора в нашем веб-приложении, я использую следующие методы:

  • Хеширование паролей: Пароли пользователей хешируются перед сохранением в базе данных, чтобы обеспечить безопасность.
  • JSON Web Tokens (JWT): После успешной авторизации администратора, на клиентской стороне генерируется JWT токен, который используется для аутентификации при последующих запросах.
  • Защищенные маршруты: Я определил защищенные маршруты, которые требуют авторизации. Если пользователь не авторизован или у него нет прав администратора, он будет перенаправлен на страницу входа.

Реализация безопасной авторизации позволяет нам защитить административные функции приложения от несанкционированного доступа.

Авторизация и восстановление пароля

В моём веб-приложении предусмотрена безопасная авторизация для администратора. Для входа в административную панель, администратор должен указать свой уникальный пароль, который хранится в зашифрованном виде в базе данных. Пароль не доступен даже администраторам системы для обеспечения максимальной безопасности.

Если администратор забыл свой пароль, предусмотрен механизм восстановления пароля. На странице входа makarovgallery.ru/login есть ссылка "Забыли пароль?", которая ведет на страницу восстановления. Для восстановления пароля, администратор должен ввести свой заранее зарегистрированный адрес электронной почты. После этого на указанный адрес будет отправлено письмо с инструкциями по восстановлению пароля.

При восстановлении пароля, система проверит введенный адрес электронной почты и если он совпадает с зарегистрированным в базе данных, администратор получит ссылку для создания нового пароля. После изменения пароля, старый пароль будет недействителен, и администратор сможет войти в административную панель с использованием нового пароля.

Этот механизм восстановления пароля обеспечивает безопасность и позволяет администраторам восстановить доступ в случае утери пароля.

Технологии

Для создания моего уникального веб-приложения, я использовал передовые технологии для обеспечения его стабильной и производительной работы. Проект состоит из трех основных частей: бэкенд, фронтенд и база данных.

Бэкенд: Symfony

Для реализации мощной и надежной серверной части приложения, я выбрал Symfony - высокопроизводительный PHP-фреймворк. Symfony предоставляет множество инструментов и компонентов, которые позволяют мне быстро разрабатывать сложные функции, обрабатывать запросы, управлять базой данных и обеспечивать безопасность данных.

Для реализации красивого и интуитивного пользовательского интерфейса, я обратился к мощному CSS-фреймворку Tailwindcss. С его помощью я создал современный и адаптивный дизайн моего приложения, который отлично выглядит на различных устройствах и экранах.
